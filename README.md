안녕하세요, Kim's Farm 개발자 최은일입니다.
Kim's Farm은 프로그래머인 저와 아트 디렉터 총 두 명이 함께 개발한 UE5.3 BP Only 프로젝트입니다.
솔로플레이 시나리오 게임이며 생존, 모험, 미션, 스토리(멀티엔딩)가 주된 컨텐츠입니다.

핵심이 되는 스토리 진행은 TimeLimit과 Progress 시스템을 사용하여 구현하였습니다.
다이얼로그는 Data Sheet에셋에서 추출해 인게임에서 Library 변수로 보관, 인덱스를 사용해 선택적으로 Library를 열람해 사용하는 방식을 채택했습니다.
아이템 수집과 미션(노동)은 자체 제작한 BP_Item을 사용하여 구현하였습니다. 퍼포먼스를 고려하여 Spawn과 Destroy를 최대한 일괄적으로 처리하였습니다.
NPC들의 Behavior Tree 와 AnimBP도 모두 자체 제작하였습니다. 목표 액터 부근의 도달 가능한 적절한 위치를 추출하여 Move To로 Behavior Tree를 구현하였고 IK를 맞춤형으로 스크립팅하여 자연스러운 움직임을 연출했습니다.
거의 모든 스크립팅이 자체 제작이며, 때문에 작업 기간이 오래 걸렸지만 그만큼 언리얼 엔진과 BP 스크립팅에 큰 숙련도를 얻었습니다. 물론 항상 자체 제작을 고집하는 건 아닙니다. 필요시 적절한 플러그인과 패키지를 활용해야 된다고 생각하며, 이러한 경우에도 커스터마이징이 필요하기 때문에 오히려 그 제작 과정을 직접 한 번 해 볼 필요가 있다고 느꼈을 뿐입니다.



여담으로, 2인 제작 게임이기 때문에 멀티 롤을 맡아야 했습니다. 비쥬얼 스크립트 이외에 맡았던 역할들을 설명해 드리겠습니다.

1. 게임 기획(스토리, 인물, 게임 진행, 기믹, 월드 설계 등)
2. 대사 작성 및 번역, 음성 대사(TTS) 추출.
3. 위젯 프레이밍
4. 베타 테스트 3회 진행(Q/A, 피드백 적용)
5. Perforce(SourceControl) 관리
6. BGM과 효과음 제작
7. 성능 최적화 및 패키징
8. 스팀 출시(SteamWorks 문서 작업 및 최적화)
